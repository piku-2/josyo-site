## 作品概要

指定した時刻に天気予報を自動で投稿してくれる Discord ボットです。傘が必要な日は教えてくれます。

## 使用した技術・ツール

- Python
- discord.py
- OpenWeather API

## 工夫した点

降水確率に応じてメッセージの文面を変え、人が話しているような自然な通知を目指しました。
